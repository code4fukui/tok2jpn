# tok2jpn
日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

Toki Ponaの辞書に日本語の翻訳が付いています。

## デモ
辞書は [https://code4fukui.github.io/tok2jpn/](https://code4fukui.github.io/tok2jpn/) でオンラインで利用できます。

## 機能
- トキ・ポナ - 日本語辞書
- 「linja suwi」フォントを使用してトキ・ポナの文字を表示
- ChatGPTによる辞書データを動力源としている

## データ/ API
辞書データは、このリポジトリの `tok2jpn.csv` ファイルに格納されています。
